## Requirements

  - Node v8.0.0+
  - Redis v2+

## Install

```
git clone https://github.com/dannycroft/metricio.git
```

```
npm i && npm run start
```

## Test

```
npm run test
```

## Coverage

```
npm run coverage
```

Open your browser at `http://localhost:3000`
