Customisation

Everything you need can be found in src:

Path Description
/src/dasboards Where all dashboards should be created. A default dasboard is already available for you to see. See dashboards for more infomation.
/src/widgets Contains all currently available widgets. You can create your own or use the existing ones. See widgets for more infomation.
/src/jobs Contains all currently available jobs. You can create your own or use the existing ones. See jobs for more infomation.
